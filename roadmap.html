The Curriculum Journey

Month 1 (Days 1 - 30)

Foundation & Fundamentals

Laying the groundwork for your technical and cognitive skills.

  • Python Basics: Variables, Data Types, Operators, Type Casting, I/O Operations, and String Manipulation.
  • Control Flow & Functions: If/Else constructs, For/While loops, Break/Continue, defining custom functions, and Lambda expressions.
  • Data Structures (Intro): Deep dive into Lists, Tuples, Dictionaries, and Sets (mutability vs immutability).
  • Aptitude (Quantitative): Number Systems, Percentages, Profit & Loss, Simple & Compound Interest, Ratio & Proportion.
  • AI Focus (Week 1-4):
    - Mastering ChatGPT for code explanation & error debugging.
    - GitHub Copilot fundamentals: Autocomplete, inline chat, and code generation.
Month 2 (Days 31 - 60)

DSA & Logical Reasoning

Sharpening your problem-solving abilities for technical rounds.

  • Python OOPs & Advanced: Classes, Objects, Inheritance, Polymorphism, Encapsulation, Exception Handling, and File I/O.
  • DSA Foundations: Time/Space Complexity (Big O), Linear vs Binary Search, Bubble/Merge/Quick Sort algorithms.
  • Linear DSA: Implementing Stacks, Queues, and Linked Lists (Singly & Doubly) from scratch.
  • Aptitude (Logical): Blood Relations, Seating Arrangements, Syllogism, Coding-Decoding, and Number Series.
  • AI Focus (Week 5-8):
    - Using AI for automated unit testing and refactoring.
    - Image generation AI (Midjourney/DALL-E) for project asset creation.
Month 3 (Days 61 - 90)

Advanced Algorithms & Placement Prep

Company-specific training and mastering the interview.

  • Advanced DSA: Binary Trees, BST, Graph Traversals (BFS/DFS), and Intro to Dynamic Programming (Memoization vs Tabulation).
  • Aptitude (Verbal & Advanced): Reading Comprehension, Sentence Correction, Data Interpretation, and Time/Speed/Distance problems.
  • Targeted Company Prep:
    - TCS NQT: Specific coding patterns and trait assessments.
    - Infosys SP/SE: High-difficulty pseudo-code and puzzles.
    - L&T: Core engineering and logic-based rounds.
  • AI Focus (Week 9-12):
    - Using AI for Resume Parsing (ATS bypass strategies).
    - Conducting AI-driven mock technical and HR interviews.
